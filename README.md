  <h2 class="text-2xl font-semibold mb-4 mt-10">Chapitre 2 : Grammaire fondamentale</h2>
  <div class="space-y-4">
    <h3 class="text-xl font-bold">1. Verbes SER et ESTAR (être)</h3>
    <p>
      En espagnol, il existe deux verbes pour exprimer "être" : <strong>ser</strong> (identité, origine) et <strong>estar</strong> (état, position). Ce chapitre est essentiel pour éviter les erreurs de base.
    </p>

    <table class="w-full table-auto border border-gray-400">
      <thead class="bg-gray-200">
        <tr>
          <th class="border px-4 py-2">Sujet</th>
          <th class="border px-4 py-2">SER</th>
          <th class="border px-4 py-2">ESTAR</th>
        </tr>
      </thead>
      <tbody>
        <tr><td class="border px-4 py-2">Yo</td><td class="border px-4 py-2">soy</td><td class="border px-4 py-2">estoy</td></tr>
        <tr><td class="border px-4 py-2">Tú</td><td class="border px-4 py-2">eres</td><td class="border px-4 py-2">estás</td></tr>
        <tr><td class="border px-4 py-2">Él/Ella</td><td class="border px-4 py-2">es</td><td class="border px-4 py-2">está</td></tr>
        <tr><td class="border px-4 py-2">Nosotros</td><td class="border px-4 py-2">somos</td><td class="border px-4 py-2">estamos</td></tr>
        <tr><td class="border px-4 py-2">Vosotros</td><td class="border px-4 py-2">sois</td><td class="border px-4 py-2">estáis</td></tr>
        <tr><td class="border px-4 py-2">Ellos/Ellas</td><td class="border px-4 py-2">son</td><td class="border px-4 py-2">están</td></tr>
      </tbody>
    </table>

    <h3 class="text-xl font-bold mt-4">2. Quand utiliser SER ou ESTAR ?</h3>
    <ul class="list-disc list-inside">
      <li><strong>Ser</strong> : Yo soy francés. (origine)</li>
      <li><strong>Estar</strong> : Estoy cansado. (état temporaire)</li>
      <li><strong>Ser</strong> : Ella es profesora. (profession)</li>
      <li><strong>Estar</strong> : Estamos en casa. (localisation)</li>
    </ul>

    <h3 class="text-xl font-bold mt-4">3. Exercice - Complétez avec SER ou ESTAR :</h3>
